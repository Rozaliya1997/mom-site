## 🚦 Что осталось сделать — по порядку

### Шаг 1️⃣ — Вставить URL Apps Script в сайт

📖 **Файл:** `GOOGLE-SHEETS-SETUP.md` (раздел про вставку URL)

После того как ты опубликовала Apps Script как веб-приложение, у тебя есть длинная ссылка вида:
```
https://script.google.com/macros/s/AKfyc.../exec
```

Эту ссылку нужно вставить в `index.html`:

1. Открой `index.html` в любом текстовом редакторе (Блокнот, TextEdit, VS Code)
2. **Ctrl + F** → найди `PASTE_YOUR_GOOGLE_APPS_SCRIPT_URL_HERE`
3. Замени этот placeholder на свой URL (он должен остаться **внутри кавычек**)
4. **Ctrl + S** — сохрани

**Должно получиться так:**
```html
<form ... action="https://script.google.com/macros/s/AKfyc.../exec" method="POST">
```

### Шаг 2️⃣ — Быстро протестировать на Netlify (необязательно, но рекомендую)

📖 5 минут, чтобы убедиться, что форма реально отправляет заявки до того, как настраивать GitHub.

1. Открой https://app.netlify.com/drop
2. Перетащи папку `site-classic` целиком на страницу
3. Получишь временную ссылку типа `https://random-12345.netlify.app`
4. Открой → пролистай до формы → заполни как тестовый клиент → отправь
5. Проверь:
   - В Google Таблице появилась новая строка ✅
   - На почту пришло письмо ✅

Если работает — переходи к GitHub. Если не работает — пиши, разберёмся.

### Шаг 3️⃣ — Опубликовать на GitHub Pages

📖 **Файл:** `GITHUB-PAGES-SETUP.md` (полная пошаговая инструкция)

Краткая выжимка:

1. **GitHub:**
   - Создать публичный репозиторий `mom-site`
   - Загрузить **всё содержимое** папки `site-classic` (включая `CNAME`)
   - **Settings → Pages → Source: `main` branch** → Save

2. **Namecheap (Advanced DNS):**
   - Удалить все стандартные записи
   - Добавить 5 новых:
     ```
     A     @     185.199.108.153
     A     @     185.199.109.153
     A     @     185.199.110.153
     A     @     185.199.111.153
     CNAME www   твой-username.github.io
     ```

3. **GitHub → Settings → Pages → Custom domain:** `havizovavenera.com` → Save

4. Подождать 15 минут – 2 часа, пока DNS разлетятся (можно проверять на https://dnschecker.org)

5. Поставить галочку **Enforce HTTPS** (появится, когда DNS дойдут)

6. Готово 🎉
