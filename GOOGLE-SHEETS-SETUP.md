# 📊 Подключение формы к Google Таблице

Эта инструкция настроит сайт так, что:
- ✅ Все заявки автоматически попадают в Google Таблицу (можно фильтровать, экспортировать)
- ✅ На указанную почту приходит красивое письмо-уведомление о каждой новой заявке
- ✅ Всё бесплатно, без сторонних сервисов и подписок

**Важно:** настраивать нужно из аккаунта Google. Удобнее всего — твоего (Розы), потому что предполагает несколько технических действий. Письма всё равно будут приходить на почту мамы (`hafizovavenera@mail.ru`).

Вся настройка занимает ~10 минут.

---

## Шаг 1. Создать Google Таблицу

1. Зайди на https://sheets.google.com
2. Нажми **«+» (Создать пустую таблицу)**
3. Назови её, например: **«Заявки с сайта — Венера Хафизова»**
4. Оставь открытой — она понадобится на следующем шаге

Заголовки колонок добавлять **не нужно** — скрипт сам их создаст и оформит при первой заявке.

---

## Шаг 2. Открыть редактор Apps Script

1. В таблице открой меню **Расширения** → **Apps Script**
2. Откроется новая вкладка с редактором кода
3. Удали весь код, который там по умолчанию (`function myFunction() { ... }`)
4. Открой файл **`google-apps-script.gs`** (он лежит рядом с этой инструкцией) в любом текстовом редакторе или прямо в Claude
5. Скопируй **весь** код из него и вставь в редактор Apps Script

### Проверь одну строку

В самом верху скрипта найди строку:

```javascript
const NOTIFICATION_EMAIL = 'hafizovavenera@mail.ru';
```

Если хочешь, чтобы письма приходили на другую почту (или дополнительно на свою) — измени её. Можно через запятую указать несколько: `'mom@mail.ru, roza@gmail.com'`.

Сохрани файл: **Ctrl + S** (или **⌘S** на Mac). Назвать проект можно как угодно — например, «Форма заявок».

---

## Шаг 3. Опубликовать скрипт как веб-приложение

Это даст ссылку, на которую сайт будет отправлять заявки.

1. В правом верхнем углу редактора нажми синюю кнопку **«Развернуть»** → **«Новое развёртывание»**
2. Слева от поля «Тип» нажми на ⚙️ **(шестерёнку)** → выбери **«Веб-приложение»**
3. Заполни:
   - **Описание:** `Форма заявок с сайта` (необязательно)
   - **Запуск от имени:** `Я (твой@email.com)`
   - **Кто имеет доступ:** ⚠️ **«Все»** ← это важно, иначе сайт не сможет отправлять
4. Нажми **«Развернуть»**
5. Появится окно с просьбой авторизоваться:
   - **«Authorize access»** → выбери свой Google-аккаунт
   - Появится предупреждение **«Google не проверило это приложение»** — это нормально (это твой собственный скрипт)
   - Нажми **«Дополнительные»** → **«Перейти к проекту… (небезопасно)»**
   - Разреши доступ к таблицам и отправке писем — нажимай **«Разрешить»**
6. Появится **URL веб-приложения** — он выглядит так:
   ```
   https://script.google.com/macros/s/AKfyc...очень-длинная-строка.../exec
   ```
7. **Скопируй этот URL** — он понадобится в следующем шаге!

---

## Шаг 4. Вставить URL в сайт

1. Открой файл `index.html` в любом текстовом редакторе
2. Нажми **Ctrl + F** (поиск) и найди строку:
   ```
   PASTE_YOUR_GOOGLE_APPS_SCRIPT_URL_HERE
   ```
3. Замени её на URL, который скопировала на Шаге 3
4. Должно получиться так:
   ```html
   <form ... action="https://script.google.com/macros/s/AKfyc.../exec" method="POST">
   ```
5. Сохрани файл

Если сайт уже опубликован на Netlify — заново перетащи папку с обновлённым файлом, и сайт обновится.

---

## Шаг 5. Проверить, что всё работает

### Тест 1 — из самого скрипта (без открытия сайта)

В редакторе Apps Script:
1. Сверху найди выпадающий список (рядом с кнопкой «Выполнить») и выбери функцию **`testSubmit`**
2. Нажми **«Выполнить»**
3. Открой свою Google Таблицу — там появится тестовая строка ✅
4. Открой почту, указанную в `NOTIFICATION_EMAIL` — там должно быть тестовое письмо ✅

### Тест 2 — с самого сайта

1. Открой опубликованный сайт
2. Заполни форму как настоящий клиент и отправь
3. Проверь:
   - В Google Таблице появилась новая строка
   - На почту пришло письмо

Если всё работает — поздравляю, форма настроена! 🎉

---

## Что делать, если не работает

### «На почту ничего не приходит»
- Проверь папку **Спам** — первые письма от Apps Script могут попасть туда. Один раз отметь как «Не спам», и дальше пойдут в основной ящик
- Проверь, что в строке `NOTIFICATION_EMAIL` нет опечаток
- Проверь дневной лимит: Google разрешает с обычного аккаунта **до 100 писем в день** через Apps Script. Для контактной формы этого хватит надолго

### «Заявки в таблицу не записываются»
- Проверь Шаг 3, пункт 3: точно ли стоит «Кто имеет доступ: **Все**»?
- Открой редактор Apps Script → меню **«Выполнения»** в левом меню. Там видны все запуски и ошибки
- Запусти `testSubmit` ещё раз и посмотри, что в логе

### «Я обновила скрипт, но изменения не применились»
- Apps Script кэширует развёртывания. После изменения скрипта нужно сделать **«Развернуть» → «Управление развёртываниями»** → выбрать существующее → значок карандаша → «Версия» → **«Новая версия»** → **«Развернуть»**
- ⚠️ URL после нового развёртывания **не меняется** (если ты редактируешь существующее развёртывание)

## Что лежит в Google Таблице

Каждая заявка — одна строка:

| Дата и время | Имя | Телефон | Email | Тип бизнеса | Задача | Удобное время |
|---|---|---|---|---|---|---|
| 10.05.2026 14:30 | Иван | +7 999... | ivan@... | ИП на УСН | ... | будни после 18 |

Можно делать что угодно: сортировать, фильтровать, отмечать какие отработала, добавлять колонку «Статус», экспортировать в Excel и т.д. — это обычная Google-таблица.

**Совет:** добавь колонку «Статус» справа (Новая / В работе / Закрыта) и фильтруй по ней — получится мини-CRM прямо в таблице.
